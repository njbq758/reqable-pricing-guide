## Reqable 是什么

简单说，Reqable 是一款跨平台的 HTTP 开发与调试工具，支持 Windows、macOS、Linux、iOS、Android，五大平台全覆盖。

它的核心定位就是「抓包 + 接口测试」一站式完成：

- **流量抓包调试**：支持 HTTP/1.x、HTTP/2、HTTP/3（QUIC）、WebSocket，HTTPS 解密，支持手机协同抓包
- **API 测试管理**：创建请求、环境变量、脚本测试、批量编辑，功能对标 Postman
- **请求响应对比**：内置可视化 Diff 工具，快速找差异
- **自由重写与断点**：规则重写、断点拦截、Python 脚本，高度可定制
- **多端协同**：手机流量一键扫码转发到电脑，不用手动配 Wi-Fi 代理

它基于 Flutter 和 C++ 开发，所以启动速度和运行流畅度远超 Postman 那种浏览器套壳的方案——有测试数据显示，同样的设备上 Reqable 启动只需约 1.5 秒，而 Postman 需要 10 秒左右。
